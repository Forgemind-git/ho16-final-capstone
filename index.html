Final Capstone — AI over Google Workspace

Week 6 M19 · Plan, Build, Ship Peer-reviewed Capstone
Done when
A deployed AI solution over your Google Workspace (Sheets · Gmail · Calendar), using the whole course · your final project.
What you build

Your final capstone — a deployed AI solution that puts Claude to work over your Google Workspace (Google Sheets, Gmail and Google Calendar), combining the pieces you built across the whole course: a research agent, a grounded assistant over your docs, a scheduled automation, and a live dashboard, all working on one real problem. Ship the deployed solution plus a story-driven README (problem → approach → tools used across the course → result), a 3–5 min demo, and a "concepts used" checklist. Each sample is a complete blueprint you can open, run locally, and deploy.

Pick a problem statement

Each sample is a real, hosted build — a full feature map and the course pieces already wired together. Open one live, or bring your own capstone (get it approved first).

1AI Ops Command Center

Run your operations from one AI cockpit. An AI ops command center reads your Google Sheets (orders, inventory) and Google Calendar, shows a live dashboard, flags what needs attention, and each morning drafts the daily status email as a Gmail draft.

✓ KPIs from your Sheets, a Calendar strip, an alerts feed and an AI-drafted daily Gmail update ready to send Open live sample
2AI Sales Intelligence

Your pipeline is a messy Google Sheet. An AI sales-intelligence platform reads that sheet, scores each deal, and drafts tailored follow-up emails in Gmail. Success: a pipeline board straight from your sheet with AI outreach queued and ready to send.

✓ pipeline board from your Sheet, AI deal scores and per-deal AI-drafted Gmail follow-ups Open live sample
3AI Content Engine

Research, drafting and scheduling are scattered. An AI content engine reads a content-calendar Google Sheet, drafts the posts, and builds a Google Slides deck of the week's plan.

✓ a content calendar from your Sheet, AI-drafted posts and a generated Slides deck preview Open live sample
4AI Finance Copilot

You never know where the money goes. An AI finance copilot reads transactions from a Google Sheet, categorises spend, surfaces insights, and emails you a weekly summary via Gmail.

✓ spending KPIs and charts from your Sheet plus an AI-drafted weekly finance Gmail summary Open live sample
5AI Support Co-Pilot

Your support tickets live in a Google Sheet and answers are slow. An AI support co-pilot triages the tickets, suggests replies grounded in your docs, and sends them via Gmail — with trends visible to leadership.

✓ a ticket dashboard fed by your Sheet, AI-suggested replies ready to send and trends for leadership Open live sample

Use your Claude.ai Pro plan wisely
Runs on your machine, then deploys
  • This is the final deployed AI solution — a full-stack app you run on localhost first. This GitHub Pages page is static and can't run the backend or the AI calls, so it only hosts this landing page and the README.
  • Follow the README to run it locally — e.g. npm install then npm run dev, with your .env and database configured.
  • Taking it to a live public URL is covered once in the course — see the Week-6 deploy walkthrough video.
Subscription-first — no API key
  • The primary path is built with your normal Claude.ai login (Pro or Team) — no API key needed.
  • Use the example prompt in each sample — it's already written and tested. Don't reinvent it.
  • Start a new chat when you switch tasks — long chats re-read every earlier message and use your limit faster.
  • Using Claude Code or Cowork? This repo's CLAUDE.md makes Claude follow these rules automatically, and SKILL.md is a reusable "token-wise" skill.
