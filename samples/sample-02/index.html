Sales Intelligence Platform

Sales teams waste hours manually researching companies and writing personalised emails from scratch. This capstone builds an AI-powered sales intelligence hub that enriches leads, scores them, drafts personalised outreach, and tracks a deal pipeline — all using Claude.ai subscription features with no API key.

No API key required. This solution is built entirely on your Claude.ai Pro or Team subscription — using Projects, Artifacts, Skills, and MCP integrations.

Architecture — Claude.ai Features Map

Every component of the Sales Intelligence Platform maps to a Claude.ai feature. No CRM subscription, no API bills — just your Claude.ai account.

Project

Sales Intelligence Hub

One Claude.ai Project holds all your leads, enrichment data, and email drafts. Conversations are persistent — your CRM log lives here.

Skill

Sales Assistant Persona

Project Instructions define Claude as a Senior Sales Intelligence Analyst — enriching leads, scoring them 1–10, and always citing sources for intel.

Artifact

Pipeline Kanban Board

Claude generates a live HTML Kanban Artifact showing deals by stage (Prospect → Qualified → Proposal → Closed). Drag cards between columns directly in the browser.

File Upload

Lead Lists & Research

Upload a CSV of target accounts or paste company profiles. Claude reads the file and enriches every lead with industry, size, pain points, and decision-maker insights.

MCP

Live Company Data

Connect a web search MCP tool so Claude can look up real company news, funding rounds, and recent announcements before drafting outreach.

Cowork

Team Sales Room

Share the Project with your sales team. AEs, SDRs, and managers all work from the same enriched lead database and draft emails together.

Build It Step by Step

  1. Create the Project Go to claude.ai → Projects → New Project. Name it "Sales Intelligence Platform". This becomes your persistent CRM workspace.
  2. Set the Sales Assistant System Prompt In Project Settings → Instructions, paste the Sales Intelligence Analyst prompt (see below). Claude will now enrich, score, and draft emails as a trained sales analyst on every conversation.
  3. Upload Your Lead List Upload a CSV or paste a list of company names + contacts into the Project. Claude ingests the list and stores it as shared knowledge — every subsequent conversation can reference these leads.
  4. Run Lead Enrichment Use the Company Enrichment Prompt. For each company name, Claude returns a structured profile: industry, estimated size, likely pain points, buying triggers, and a lead score 1–10 with reasoning.
  5. Draft Personalised Outreach Emails After enriching a lead, paste the Email Drafting Prompt. Claude writes a personalised cold email that references specific company facts from the enrichment — not a generic template.
  6. Generate the Pipeline Kanban Artifact Paste the Kanban Generation Prompt. Claude creates a fully interactive HTML Artifact with all your leads sorted into stages. You can move cards between columns and mark deals as won or lost directly in the Artifact.
  7. Connect Live Research via MCP In Claude.ai Settings → Integrations, enable a web search or Brave Search MCP. Then ask Claude to research a company before enriching it — Claude will pull real news, funding, and job postings to ground the profile.
  8. Invite Your Sales Team (Cowork) Share the Project with your team. Each member can run their own enrichment and email sessions, and managers can review all conversations in one place — the same leads, same scoring rubric, consistent output.

Prompts & Configs

Project Instructions (System Prompt)
You are a Senior Sales Intelligence Analyst for this team.

Your responsibilities:
1. ENRICH company profiles: given a company name, infer industry, estimated headcount, likely tech stack, key pain points, buying triggers, and decision-maker titles. Return structured JSON.
2. SCORE leads 1–10 with a brief rationale: 8–10 = strong ICP fit, 5–7 = worth nurturing, 1–4 = low fit.
3. DRAFT personalised outreach emails: use the enriched data to reference specific company facts — never send generic templates.
4. TRACK pipeline stages: Prospect → Qualified → Proposal → Closed Won / Closed Lost.

Always cite your reasoning. If you used web search results, quote the source.
Format all enrichment output as JSON before the human-readable summary.
Company Enrichment Prompt
Enrich this lead:

Company Name: [COMPANY NAME]
Contact Name: [CONTACT NAME] (optional)
Contact Title: [TITLE] (optional)

Return a JSON block first:
{
  "company": "...",
  "industry": "...",
  "estimated_size": "...",
  "hq_location": "...",
  "likely_tech_stack": [...],
  "key_pain_points": [...],
  "buying_triggers": [...],
  "decision_maker_titles": [...],
  "lead_score": 8,
  "score_rationale": "..."
}

Then write a 3-sentence sales brief explaining why this company is (or isn't) a strong prospect for us.
Personalised Email Drafting Prompt
Using the enrichment profile for [COMPANY NAME], write a personalised cold outreach email.

Requirements:
- Subject line: compelling, specific to the company (not generic)
- Opening line: reference one specific fact from the enrichment (pain point, recent news, or trigger)
- Body: 3 sentences max — one problem statement, one value prop tied to their specific situation, one CTA
- CTA: request a 20-minute call, include a Calendly-style placeholder link
- Tone: direct, peer-to-peer, no buzzwords or fluff
- Sign-off: include name and title placeholders

Output: Subject + Email body (under 150 words total)
Pipeline Kanban Artifact Prompt
Generate a self-contained HTML Artifact: an interactive Sales Pipeline Kanban Board.

Columns: Prospect | Qualified | Proposal | Closed Won | Closed Lost

Use the following leads (add them as cards in their current stage):
[PASTE YOUR LEAD LIST WITH STAGES HERE]

Each card should show:
- Company name (bold)
- Lead score badge (colour-coded: green 8–10, yellow 5–7, red 1–4)
- Contact name + title
- One-line pain point or trigger

Styling: #1E40AF primary colour, clean card shadows, responsive grid.
Make cards draggable between columns using vanilla JS drag-and-drop.
Include a summary bar at the top: total leads, avg score, # in each stage.
Lead Scoring Batch Prompt
I'm uploading a list of [N] companies. Score each one 1–10 based on ICP fit for a B2B SaaS company targeting [YOUR ICP DESCRIPTION].

For each company return:
- company_name
- industry
- estimated_size
- lead_score (1–10)
- top_reason (one sentence)

Return as a markdown table, then a JSON array below the table.

Prioritise by score (highest first).

Done When

  • Project created in Claude.ai with the Sales Intelligence Analyst system prompt active
  • Lead list uploaded (CSV or pasted) and accessible to all Project conversations
  • At least 5 companies enriched with JSON output including lead scores
  • At least 3 personalised outreach emails drafted (each referencing specific company facts)
  • Pipeline Kanban Artifact generated and functional in the browser with draggable cards
  • MCP web search connected — at least 1 enrichment uses real-time company data
  • Batch scoring run on 10+ leads with output as a sortable table
  • Team invited and at least 1 collaborator has run their own enrichment session
  • Demo: company name → enrichment → email draft → pipeline card — end-to-end in one conversation
