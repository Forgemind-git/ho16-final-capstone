# Sample 04 — Personal Finance Copilot

> **Problem Statement:** You never know where your money goes. Build and deploy a personal-finance copilot.

## Your Mission

Design and deploy a solution that solves this problem using Claude.ai subscription features.
You do NOT need to write code — use Claude Projects, Cowork agents, Skills, and Artifacts.

## Components to Build

Work through each component below. Check it off when done.

- [ ] **Claude Project: grounded on bank exports**
  - Export your bank statement (CSV or PDF) and upload it to a Claude Project
  - Write a system prompt that positions Claude as your personal finance analyst
  - Test: can Claude answer questions about your spending accurately?

- [ ] **Cowork agent: monthly categorisation**
  - Design a Cowork agent that reads your transactions and categorises them
  - Define your spending categories (Food, Transport, Subscriptions, etc.)
  - Test the agent with at least one month of real data

- [ ] **Skill: budget vs actual analyser**
  - Create a Skill that compares your planned budget against actual spending
  - Define the input format (budget table + transactions) and output format (variance report)
  - Write and test the Skill

- [ ] **Artifact: spending breakdown chart**
  - Ask Claude to generate a spending breakdown chart Artifact
  - It should visualise spending by category for the month
  - Iterate until the chart is clear and actionable
